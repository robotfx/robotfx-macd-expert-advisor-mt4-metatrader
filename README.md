<div style="text-align: center;"><h3><a href="#" id="Functions" name="Functions" title="MACD EA Functions">MACD main functions</a></h3></div><ul>
<li>Trade following the <strong>MACD trend</strong> and open trades at <strong>MACD crossover signals</strong> (trend-following can be disabled if desired)</li>
<li>Use <strong>Stochastic oscillator</strong> overbought/oversold confirmation to enter and exit trades accurately</li>
<li>Automatically close trades and open new positions when the trend reverses</li>
<li>Set stop-loss, take-profit, trailing stops, break-even, and other risk management options</li>
</ul>
<p>
  The <strong>MACD EA for MetaTrader</strong> is designed for traders familiar with the <em>Moving Average Convergence Divergence (MACD)</em> indicator. It combines the power of MACD trend detection with <strong>Stochastic confirmation</strong> to improve trade entry precision. This expert advisor is easy to configure and use, making it suitable for both novice and experienced forex traders who want automated trend-following strategies.
</p>

<p>
  The <strong>MACD Expert Advisor for MetaTrader</strong> ensures trades follow the correct trend by using the same <strong>MACD settings</strong> that the trader uses to determine trade entries and exits. These settings can be customized for each currency pair. Trading signals are generated by the <strong>MACD crossover</strong> between the main line and the signal line.
</p>
<p>
  Following the MACD trend, the EA opens <strong>sell trades</strong> when the MACD main line crosses below the signal line, and <strong>buy trades</strong> when the main line crosses above the signal line. <strong>Stop-loss</strong> and <strong>take-profit</strong> levels are applied immediately after opening each trade, making this expert advisor fully compatible with <strong>ECN brokers</strong> and optimized for low-latency trading.
</p>

<li>UseStochConfirmation - if set to true the EA will consider buying/selling only if the Stochastic is oversold/overbought</li>
<li>Trading Settings - using this settings the trader has the option to set the lotsize of the trades, the stoploss and the takeprofit as well as the trailing stop (breakeven) and the trailing step</li>
<li>CloseAtOppositeSignal - if set to true the EA will close the opened trades when an opposite trading signal appears, according to the EA’s settings (MACD crossover, Stochastic cross etc). In other words, if there’s an opened buy and a sell signal occurs, the EA will close the opened buy trade</li>
<li>ReversedSignal - instructs the EA to open long trades when short (sell) signal occur, and vice-versa</li>
<li>EA Settings - using this settings the trader can choose the maximum number of trades allowed to be opened by the expert advisor, the slippage and the magic number, in case there’s another EA used to manage the trades opened by this expert advisor</li>
</ul><p>
  Traders can also use the <strong>Stochastic indicator</strong> to confirm trade entries and exits. In the example chart above, using default settings, the <strong>MACD EA</strong> opened three trades (maximum allowed) during an uptrend. The MACD main line crossed above the signal line while the Stochastic was in the oversold zone, resulting in two trades closed at take-profit and one closed via trailing stop.
</p>
<p>
  To improve risk management, the <strong>MACD Expert Advisor</strong> allows the trader to:
</p>
<ul>
<li>Limit the total number of open trades</li>
<li>Automatically close trades when opposite signals occur</li>
<li>Use stop-loss, take-profit, trailing stop, and break-even for each trade</li>
</ul>
<p>
  These features make the <strong>MACD EA</strong> suitable for trend-following strategies while giving the trader flexibility to confirm trades using the <strong>Stochastic oscillator</strong> or other indicators.
</p>
